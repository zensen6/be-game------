1. 블랙잭 룰

블랙잭은 플레이어가 딜러에게 카드를 한 장씩 받아 모든 카드의 숫자 합이 21에 가장 가까운 사람이 이기는 게임이다.
<br>
숫자 합이 21을 초과할 경우 게임에 지게 되며, 플레이어는 자신의 숫자 합을 고려하여 딜러가 제시하는 랜덤의 카드를 받을지 받지 말지 결정하게 된다.
<br>
<br>
